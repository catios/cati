# Cati package manager
cati is a package manager for GNU/Linux/UNIX systems. cati is written in python

## installation
to compile and install cati on your system, do the following steps:

```bash
cp /path/to/cati

make
sudo make install
```

now you can run cati in termianl:

```bash
cati
```
