Module helpers.hash
===================
Some helper functions for hashing

Functions
---------

    
`calc_file_md5(filepath)`
:   gets filepath and calculates md5 sum of that
    
    Args:
        filepath (str): filepath you want to calculate hash of that
    
    Returns:
        str: calculated hash
